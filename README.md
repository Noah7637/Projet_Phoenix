# Projet A2 Phoenix 2026

## Prérequis

- [PHP](https://www.php.net/) (version recommandée : 8.1+)
- [Composer](https://getcomposer.org/)

## Installation

1. Clonez le dépôt :
    ```bash
    git clone https://github.com/<votre-utilisateur>/Projet_A2_Phoenix_2026.git
    cd Projet_A2_Phoenix_2026
    ```

2. Installez les dépendances avec Composer :
    ```bash
    composer install
    ```

## Lancement du projet

- Pour démarrer le serveur local :
  ```bash
  php -S localhost:8000 -t public
  ```

## Configuration

- Copiez le fichier `.env.example` en `.env` et configurez vos variables d'environnement.
